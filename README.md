# Sandpile

探索前端开发最优架构，基于MD笔记在线阅读编辑为基础需求进行开发。

## 系统

- [x] 用户登录、注册
- [x] 保持用户登录状态(基于Token实现)
- [x] 用户权限管理
- [x] 国际化
- [ ] 后台管理系统
  - [ ] 支持多语言在线配置，附加预览窗口，支持词语定位、高亮，支持修改后实时生效
  - [ ] 支持用户、角色、权限管理
  - [ ] 支持埋点监控

## Markdown笔记模块

- [x] 笔记项目：每个笔记项目内可包含多个笔记文件及文件夹
- [x] 笔记文件：MD格式文本文件
- [x] 解析器：解析MD文本为HTML，并根据多级标题生成导航菜单
- [x] 支持在线编辑MD笔记，同时预览解析后的内容
- [x] 支持上传MD文件或包含MD文件的文件夹

## 3D模块

- [x] DEMO1：以图片为模板的平面粒子，在引力驱动下运动
- [ ] DEMO2：完全基于物理引擎+引力运动的星系，支持控制飞船模拟飞行
